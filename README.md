# ❌⭕ Tic-Tac-Toe Web Application

## 📌 Overview

The Tic-Tac-Toe Web Application is a modern and interactive browser-based game built using HTML, CSS, and JavaScript. It allows users to play either against another player or against an AI opponent. The application features player name customization, winner and draw detection, game reset functionality, and a responsive, user-friendly interface.

This project demonstrates core web development concepts including DOM manipulation, event handling, game logic implementation, conditional statements, and responsive UI design.

---

## 🚀 Features

* 🎮 Play in **Player vs Player** mode
* 🤖 Play against an **AI Opponent**
* 👤 Enter custom player names
* ❌⭕ Automatic turn switching
* 🏆 Winner detection
* 🤝 Draw detection
* 🔄 Reset the current game
* 🆕 Start a new game anytime
* 📱 Responsive design for different screen sizes
* 🎨 Modern and interactive user interface

## 💡 How It Works

1. Select a game mode (**Player vs Player** or **Player vs AI**).
2. Enter the player name(s).
3. Click the **Start Game** button.
4. Players take turns placing **X** and **O** on the board.
5. The game automatically detects a winner or a draw.
6. Use **Reset Game** to restart the current match or **New Game** to return to the home screen.
